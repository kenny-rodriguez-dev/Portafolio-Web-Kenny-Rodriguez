# Portafolio Web - Kenny Rodríguez

Este es mi **Portafolio Web Profesional**, un proyecto Front-End moderno desarrollado con **Angular**, enfocado en presentar de forma clara, visual e interactiva mi perfil profesional, experiencia, proyectos y formación como desarrollador.

El sitio está diseñado con una fuerte atención al **detalle visual**, **experiencia de usuario** y **buenas prácticas de desarrollo**, combinando animaciones, componentes reutilizables y un diseño completamente responsivo.

---

## Estado del Portafolio

Este portafolio está **completo y funcional**.  
Está orientado a **reclutadores y equipos técnicos** que deseen evaluar mis habilidades en desarrollo Front-End, arquitectura de componentes y diseño de interfaces modernas.

---

## Características Principales

- **Diseño Visual Avanzado y Animaciones**  
  Uso intensivo de **SCSS**, animaciones personalizadas, efectos hover y transiciones suaves para una experiencia moderna y profesional.

- **Fondo Interactivo**  
  Implementación de un fondo animado con partículas dinámicas, interacciones con el cursor, efectos de red y optimización específica para dispositivos móviles.

- **Secciones Claramente Definidas**  
  - Sobre mí  
  - Tecnologías
  - Experiencia profesional
  - Proyectos
  - Formación  
  - Información de contacto y redes profesionales

- **Componentes UI Personalizados**  
  Botones con iconos, badges tecnológicos, tarjetas informativas y componentes interactivos diseñados desde cero.

- **Iconografía Moderna**  
  Integración de **Lucide Icons** para una interfaz limpia, consistente y profesional.

- **Diseño 100% Responsivo**  
  Adaptación completa a dispositivos móviles, tablets y escritorio, con ajustes visuales específicos por breakpoint.

- **Buenas Prácticas de Front-End**  
  Código tipado, separación de responsabilidades, componentes reutilizables y enfoque en mantenibilidad.

## Tecnologías Utilizadas

- **Angular** (Standalone Components)
- **TypeScript**
- **SCSS**

## Requisitos Previos

Para ejecutar el proyecto localmente necesita tener instalado:

- **Node.js** (versión 16 o superior)
- **Angular CLI**

---

## Configuración y Ejecución del Proyecto

### 1. Clonar el Repositorio

```bash
git clone https://github.com/kenny-rodriguez-dev/portafolio-kenny-rodriguez.git
cd portafolio-kenny-rodriguez
```

#### 2. Instalar Dependencias

1.  Instale las dependencias de Node.js:

    ```bash
    npm install
    ```

2.  Ejecute el proyecto en modo de desarrollo:

    ```bash
    ng serve
    ```

La aplicación estará disponible en `http://localhost:4200`.
